Student Dashboard

Files:
- index.html — Main HTML for the dashboard
- styles.css — Responsive layout and component styles
- script.js — Small interactivity (sidebar toggle)

How to run
1. Open `index.html` in your browser (double-click or use the editor preview).
2. Resize the browser to test responsive behavior. On small screens, use the ☰ menu to open the sidebar.

Notes
- Built with CSS Grid and Flexbox for responsiveness.
- Colors and spacing are defined using CSS variables.
- This is a static layout; integrate with your backend or framework as needed.
